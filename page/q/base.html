На чем построен сайт?

Основа сайта:

  1. Bootstrap - свободный набор инструментов для создания сайтов и веб-приложений. Включает в себя HTML- и CSS-шаблоны оформления для типографики, веб-форм, кнопок, меток, блоков навигации и прочих компонентов веб-интерфейса, включая JavaScript-расширения

    2. Именно из-за использования Bootstrap некоторые браузеры не поддерживаются

  2. Сайт написан с использованием стандартов HTML5 (HyperText Markup Language, version 5) и CSS3 (Cascading Style Sheets 3)

    3. Вследствие этого пришлось отказаться от поддержки браузера Internet Explorer 11, а так же от поддержки операционных систем (семейства Windows) ниже Windows 8.1

Почему на страницах с ответами на вопросы нет оформления?

Всё просто – если Вы зашли сюда, то, скорее всего, у Вас есть проблемы с отображением страниц сайта. Чтобы проблем было меньше, мы решили не использовать современные стандарты разметки страниц, а вместо них применить устаревшие, которые точно поддерживают все браузеры
