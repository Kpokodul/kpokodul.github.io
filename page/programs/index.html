INS (Interpreter of Number Systems) - Перевод систем счисления

Лицензия: GNU GPL version 3
Операционная система: Windows / Linux
Платформа: Python (Tkinter)
Версия программы: 0.1.0 (Стабильная Графическая версия)

Функции:
  • Перевод числа из 2, 8, 10, 16 систем счисления в 2, 8, 10, 16 системы счисления
  • Запись результата всех операций в log файл
